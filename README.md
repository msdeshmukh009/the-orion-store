# [Orion store](https://the-orion-store.netlify.app/)

Buy your space merchandise from here.

# Tech Stack

- React Js
- Backend from [Mockbee](https://mockbee.netlify.app/)

# Features Added

- Home page with category section.
- Product listing page with filter section.
- Filter by
  - Price
  - Category
  - Rating
- Cart Management
- Wishlist Management
- Authentication
  - Sign-up
  - Login
  - Logout
- Search
- Single Product Page
- Loading & Alerts
- User Profile Page
- Apply Coupons.
